## 📌 Project Overview

This project focuses on building a **Digital Literacy Portfolio** as part of the Student Digital Ambassador role. The main aim is to develop awareness and practical skills needed to use digital tools effectively, safely, and professionally.

Through this project, I explored different aspects of digital literacy, including online safety, professional communication, coding practice, and building a digital presence.

The project includes multiple tasks such as:
- Creating an infographic to explain digital literacy  
- Designing a quiz to spread awareness  
- Practicing coding on platforms like Hackerrank 
- Writing professional emails  
- Developing a cybercrime case study and prevention checklist  

I also worked on building profiles on platforms like GitHub, LinkedIn, and Kaggle to create a strong digital presence.

Overall, this project combines creativity, technical skills, and real-world awareness. It helped me improve my digital knowledge and prepared me to use technology responsibly and confidently in academic and professional environments.
